# Taller – Optativa IV
## Paralelismo y Concurrencia en Java

Este repositorio contiene el desarrollo del taller de la asignatura **Optativa IV**, enfocado en el uso de **hilos en Java para paralelismo y concurrencia**.

## Contenido

### Ejercicio 1 – Paralelismo
Programa que calcula la suma de los números del 1 al 1,000,000 utilizando 4 hilos en paralelo.

### Ejercicio 2 – Concurrencia
Simulación de una cuenta bancaria compartida donde varios hilos intentan retirar dinero al mismo tiempo, utilizando sincronización para evitar inconsistencias.

### Ejercicio 3 – Preguntas Teóricas
Respuestas a preguntas sobre paralelismo, concurrencia y condiciones de carrera.
